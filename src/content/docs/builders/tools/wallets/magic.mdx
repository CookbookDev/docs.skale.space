2. **Get your API Key on Magic Dashboard**

3. **Magic Setup**
```javascript
const customNodeOptions = {
  rpcUrl: 'Your SKALE RPC',
  chainId: 12345
}

 let magic = new Magic('api_key_dedicated_wallet', { 
      network: customNodeOptions,
    });
```

4. **Native token transfer example**
```javascript
    import { Magic } from 'magic-sdk';
    import { ethers } from "ethers";

    const provider = new ethers.BrowserProvider(magic.rpcProvider);

    await magic.auth.loginWithEmailOTP({ email: 'set_user_email@gmail.com'});

    const signer = await provider.getSigner();

    const txnParams = {
      to: "address_to_send_gas",
      value: ethers.parseUnits('0.000001', 'ether'),
      gasLimit: ethers.parseUnits('0.0001', 'gwei'),
      gasPrice: ethers.parseUnits('0.0001', 'gwei'),
    };

    const tx = await signer.sendTransaction(txnParams);

    console.log("TX HASH " + tx.hash)
```
